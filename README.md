# Frontend Challenges

A collection of UI coding challenges built with React and TypeScript. Each challenge is documented and viewable in Storybook.

## Stack

- **React 19** + **TypeScript**
- **Vite** — dev server and build
- **Vitest** + **React Testing Library** — unit tests
- **Storybook** — component explorer and documentation

## Getting started

```bash
npm install
```

| Command | Description |
|---|---|
| `npm run dev` | Start the dev server |
| `npm run storybook` | Browse components in Storybook |
| `npm test` | Run unit tests |
| `npm run test:ui` | Run tests with the Vitest UI |
| `npm run build` | Production build |
